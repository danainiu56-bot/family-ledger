# 月度记账本（family-ledger）

微信里用的家庭共享记账本 H5，独立于产品营销项目。

- 线上地址（GitHub Pages）：`https://danainiu56-bot.github.io/family-ledger/book/`
- 本地调试：`python3 dev-server.py` → http://localhost:8765/book.html

> 注意：GitHub 上这个仓库**必须命名为 `family-ledger`**，否则代码里的线上链接会 404。若要改名，需同步修改 `book.html`、`scripts/pages/bookkeeping.js` 里的仓库名。

发布、Supabase 建表与多人共享说明见 [发布与共享.md](./发布与共享.md)。
